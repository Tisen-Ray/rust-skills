# Rust Skills Marketplace

Installation instructions for individual Rust skills.

## Quick Install

```bash
# Clone the repository
git clone https://github.com/huiali/rust-skills.git
cd rust-skills

# Install a specific skill (example: ownership)
claude skill install skills/rust-ownership

# Or install the main entry skill
claude skill install SKILL.md
```
